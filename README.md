# Portfolio

I have always wanted a full HTML + CSS + JavaScript portfolio and here it is. It is highly inspired by this [video](https://www.youtube.com/watch?v=Slxdo0Dqxlk).

## Code style

Every formatter and linter config in this repo (Prettier, ESLint, Stylelint, markdownlint, html-validate, EditorConfig, plus the `.vscode/` workspace settings) comes from my personal style baseline:

> **[`style_config`](https://github.com/braz9LKDI/style_config)**. Copy the configs into a new repo, merge `package.sample.json` into your `package.json`, run `npm install` and you get the same setup you see here.
